# restful-hapijs

1- npm install  
2- .env properties, default:
    HOST=localhost
    PORT=3000

    POSTGRES_HOST=localhost
    POSTGRES_PORT=5432
    POSTGRES_USER=postgres
    POSTGRES_PASSWORD=root
    POSTGRES_DATABASE=postgres
    POSTGRES_SCHEMA=public

    DB_LOGGING=false
    ENVIRONMENT=dev

3- npm run watch

4- http://localhost:3000/home (no fallback for routes yet)
